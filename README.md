# MediaWebsite

一个离线本地音视频网站。

## 功能

- 设置媒体根目录
- 读取根目录下一层分类目录中的音频和视频文件
- 以目录树方式展示分类和文件
- 点击文件后，以当前分类作为播放列表开始播放
- 支持顺序播放、随机播放、循环播放
- 支持全库播放，播放范围为根目录下所有识别到的媒体文件

## 目录扫描规则

- 支持根目录下直接放置媒体文件
- 支持 `rootDir/categoryDir/file.ext`
- 当前实现只读取一层分类目录，不继续深入更深层级

## 支持格式

- 音频：`mp3` `wav` `flac` `m4a` `aac` `ogg` `opus`
- 视频：`mp4` `mkv` `mov` `webm` `avi` `m4v`

## 启动

```bash
python3 app.py
```

浏览器访问 `http://127.0.0.1:8876`。
